CSS3_웹내비게이션(Navigation) p.267

웹페이지에서 사용자가 정보를 찾기 위한 가장 기본적인 수단으로서 내비게이션(메뉴)라고 하며, 종류에는 메인메뉴, 서브메뉴, 검색메뉴, 로그인메뉴, 사이드 메뉴, 드롭다운메뉴, 탭메뉴, 사용자의 현재 위치를 알려주는 '로케이션'메뉴 등이 있다.

내비게이션 개발은 메뉴 목록을 나열하기 위해 목록태그요소 ul, li태그를 사용하고, a태그요소로 하이퍼링크를 설정한다.

gnb : global navigation bar의 약자로서 보통 메인메뉴를 작성할 때 사용하는 명칭

lnb : local navigation bar의 약자로서 메인메뉴외에 서브메뉴나 콘텐츠 영역에서 사용되는 메뉴를 작성하고자 할 때 사용하는 명칭이다.
