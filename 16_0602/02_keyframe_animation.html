애니메이션 keyframe속성은 @keyframes속성안에서 애니메이션 내용을 작성한 후 사용자가 지정한 요소에 animation이름을 지정하여 자동으로 움직이도록 한다.

    작성법)
    @keyframes 애니메이션 이름{
      0%{실행할 서식}
      100%{실행할 서식}
    }
  • animation-name : 적용할 애니메이션 이름
  • animation-duration : 애니메이션 시간설정
  • animation-interation-count : 애니메이션 반복회수
  • animation-delay : 애니메이션 지연 시간설정
