css 소개와 문법

css는 cascading style sheets의 약자로서 html 문서의 내용에 서식을 적용하기 위한 언어로서 미국의 '에릭 메이어'라는 개발자에 의해서 개발된 언어

1. css문법(작성법)

css언어는 기본적으로 head와 head사이에 작성. 문법은 '선택자{속성:값;}으로 작성

1) 선택자(selector) - 서식을 적용하기 위한 하나의 방법

2) 속성(property) - 속성은 서식의 종류, 예를 들면 글자모양, 글자크기, 줄간격, 배경색, 가로크기 등의 서식내용

3) 값(value) - 속성을 선택하면 반드시 설정하는 내용, 예를 들면 'font-size:20px;backgroung:#f00'

2. css의 특징

css 언어는 모든 문서에 동일한 서식을 만들 수 있으며, 어느 환경에서나 사용자가 의돟ㄴ 디자인을 구현할 수 있다.

css온오눈 html태그처럼 주석을 표시할수 있는데 '/*주석내용 */'으로 작성

3. css 작성자 위치지정 방법

1) 내부 스타일 - 현재 문서에서만 스타일 서식을 적용하는 방법

2) 외부 스타일 - 별도의 css 문서를 작성 저장하여 html 문서에 연결 적용하는 방법

3) 인라인스타일 - 적용하고자 하는 태그요소안에 직접 스타일 서식을 작성하여 적용하는 방법
