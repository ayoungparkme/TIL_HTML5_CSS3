08. SVG이미지

svg : Scale Vector Graphic의 약자로서 벡터방식을 사용하여 이미지가 확대 되어도 깨지지 않고 선명하게 나오는 이미지 파일 형식을 말함.
html5에서 새로 추가된 파일 확장자로서 보통 모바일화면에서 화면해상도에 따른 이미지를 깨끗하게 보여주기 위해서 사용하며 로고등에 활용된다.

svg 만들기 : 벡터방식인 그래픽 프로그램 '일러스트'에서 디자인후 file > save as > svg이미지를 선택하여 저장한다.


svg이미지 삽입방법 2가지

첫번째 방법 - img태그를 사용하여 삽입하기

두번째 방법 - svg파일을 더블클릭하면 웹브라우저에서 이미지가 보여지는데 '소스보기'를 통해 해당 소스를 복사하여 페이지에 삽입하는 방법이 있다.
