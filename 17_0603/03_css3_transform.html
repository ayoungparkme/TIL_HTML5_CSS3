03_css3_transform(변형) - 2차원, 3차원

transform속성을 사용하여 사용자가 선택한 요소의 크기를 변경(scale)하거나, 기울기(skew), 회전(rotate), 위치 이동(translate)을 하기 위한 속성이다.

  • scale - 크기를 조정하기 위한 속성으로서 %를 소수점으로 환산하여 입력한다.
  • rotate - 회원을 위한 속성이며 각도 입력시 단위 deg를 입력한다.
  • skew - 기울기를 위한 속성이며 각도 입력시 단위 deg를 입력한다.
  • translate - 위치를 이동하기 위한 속성이며 position:relative와 비슷한 속성

1. scale - 사용자가 선택한 요소의 크기를 설정함.

2. rotate - 사용자가 선택한 요소를 회전함.

3. skew - 사용자가 선택한 요소를 기울임.

4. translate - 사용자가 선택한 요소의 위치를 이동함.(position과 비슷)
